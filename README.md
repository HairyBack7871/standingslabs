# **Standing Slabs**

A mod for Minecraft that gives you **vertical slabs for every block you love**. Standing Slabs lets you place slabs upright, giving every slab type in the game a matching vertical version. It opens up new ways to shape walls, decorate builds, and add detail that isn’t possible with normal slabs. All standing slabs fit naturally into the game, using the same textures, sounds, and behavior you already know—just turned on their side so you can build with them in fresh, creative ways. You can craft them easily in survival by **shapelessly converting any slab into its standing version**, and convert them back whenever you need.

---

## **Features**

- **Vertical variants for all slabs**
- **Survival‑friendly shapeless crafting recipes**
- **Natural integration with existing builds and resource packs**
- **Fully compatible with other Fabric mods**

## **Technical Details**

Standing Slabs is built for the **Fabric modding ecosystem**, using the Fabric Loader and Fabric API.  
The project uses **Gradle** and follows the standard Fabric project structure, making it easy to explore or extend.

For development, **IntelliJ IDEA** works perfectly.  
Open the project folder, let IntelliJ import the Gradle build, and you can run, debug, or modify the mod directly from the IDE.
